# 📊 Retail Sales Analysis Dashboard

## 🔍 Project Overview
This project analyzes retail sales data to generate business insights using Excel.

## 📁 Dataset
- 1500+ sales records
- Includes product, category, city, and pricing data

## 📊 Key Metrics
- Total Revenue
- Average Order Value
- Total Orders

## 📈 Dashboard Features
- Monthly Sales Trends
- Sales by Product & Category
- Top 3 Products Analysis
- City-wise Sales Insights

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Data Visualization

## 💡 Key Insights
- Identified top-performing products
- Analyzed monthly sales patterns
- Derived insights for better decision-making
